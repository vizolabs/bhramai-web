# BhramAI

**Status: IN PROGRESS** 🚧

> Free AI Interview Assistant by VizoLabs Syndicate

## Overview

BhramAI is a 100% free, offline AI interview assistant. No login required. No tracking. Runs locally on your machine.

## Features

- Free AI-powered interview practice
- 100% Private & Offline
- No Login Required
- Dark mode interface

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```
